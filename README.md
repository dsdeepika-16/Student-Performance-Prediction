Student Performance Prediction using Machine Learning

Project Overview

This project predicts students' exam scores using Machine Learning. The project follows a complete machine learning workflow including data preprocessing, exploratory data analysis (EDA), feature encoding, model training, prediction, and evaluation.

Dataset

- Dataset: Student Performance Factors
- Total Records: 6607
- Features: 19
- Target Variable: Exam_Score

Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

Machine Learning Workflow

- Data Cleaning
- Handling Missing Values
- Exploratory Data Analysis (EDA)
- Label Encoding
- Feature Selection
- Train-Test Split
- Linear Regression
- Model Evaluation

Model Performance

| Metric | Value |
|--------|------:|
| Mean Absolute Error (MAE) | 1.02 |
| Mean Squared Error (MSE) | 4.40 |
| R² Score | 0.689 |

 Conclusion

The Linear Regression model achieved an average prediction error of approximately **1 mark** and explained about **69%** of the variation in students' exam scores.
